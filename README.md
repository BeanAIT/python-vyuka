## Repozitář pro výuku programovacího jazyka Python

### Kroky pro práci s GIT
1. ...*Dělám změny v repozitáři (programuju)*...
1. `git fetch` updatuje stav vzdáleného repozitáře
1. `git pull origin [nazev-vetve]` načte změny ze vzdáleného repozitáře do větve [nazev-vetve]
1. `git add .` všechny změny uložím do stage
1. `git commit -m "zprava"` Vytvořím commit s textem zprava
1. `git push origin [nazev-vetve]` uložím změny do vzdáleného repozitáře do větve [nazev-vetve]


Hodit se mi může ještě:
- `git status` zobrazí stav repozitáře
- `git log` zobrazí historii commitů
- `git checkout [nazev-vetve]` přepne branch na [nazev-vetve]. Pokud větev neexistuje, použiji přepínač `-b`
